# Results

This directory contains the processed outputs used in the poster and final
report.

- `figures/`: final plots and attention visualizations.
- `tables/`: processed CSV/Markdown summaries, including full-data accuracy,
  low-data accuracy, seed stability, and epoch-level metrics.
- `logs/`: per-run `summary.csv` files exported by the training jobs. Large
  checkpoint files are intentionally omitted.

Several structured-initialization runs were extended after the initial log
export. Those final checkpoint numbers are reflected in the report/poster bar
charts and in the plotting script overrides.
